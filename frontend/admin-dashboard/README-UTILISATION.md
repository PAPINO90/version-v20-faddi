# README d'utilisation — Dashboard Admin 🔧

## Présentation
Ce document décrit de façon concise et opérationnelle les rôles, fonctionnalités et flux principaux du **Dashboard Admin**. Il vise à aider les administrateurs (Admin et SuperAdmin) et à clarifier l'expérience client liée aux actions effectuées depuis le dashboard.

## Rôles et permissions 👥
- **SuperAdmin** : accès complet — peut modifier/mettre à jour le code d'accès, créer, bloquer ou supprimer des administrateurs.


- **Admin** : gère les produits, catégories, promotions, publicités, bannières, annonces, abonnements, commandes et réclamations selon les droits.


- **Client** : parcourt la boutique, ajoute au panier, passe des commandes, s'abonne et envoie des réclamations.

## Fonctionnalités principales 🔍
- **Produits** : Ajout/édition/suppression de produits depuis le dashboard → s'affichent sur la page boutique.

- **Catégories** : Création et gestion de catégories → mises à jour visibles en boutique.

- **Promotions** : Création d'offres/promo → affichage dans la page promotions. En cliquant sur « J'en profite », le client est redirigé vers le panier.

- **Publicités flottantes** : Ajout depuis le dashboard → affichage sur la page cible configurée.

- **Bannières (header / boutique)** : Gestion des bannières visuelles et textuelles → affichage immédiat sur les emplacements définis.

- **Annonces (image / vidéo)** : Gestion des annonces et des cibles de redirection lors du clic.


- **Abonnements** : Les clients s'abonnent (cloche) en fournissant email et téléphone → listes d'abonnés accessibles au dashboard ; les admins peuvent supprimer un abonné.


- **Commandes** : Réception, traitement et mise à jour du statut des commandes (En attente → En livraison → Livré).


- **Réclamations / Retours** : Réclamations envoyées par client → réception et traitement dans le dashboard.

## Flux utilisateurs (exemples) 🔁
- Ajouter un produit : Admin → « Ajouter un produit » → remplir infos → valider → produit visible en boutique.

- Promotion client : Client clique « J'en profite » → produit ajouté au panier → finalise la commande → paiement et infos de livraison → commande transmise au dashboard.

- Suivi de commande : Client fournit numéro de téléphone → reçoit détails et statut (en attente / en livraison / livré).

- Réclamation : Client envoie une réclamation → dashboard admin reçoit → admin traite et répond.

## Gestion des accès & administrateurs 🔐
- **Modifier le code d'accès** : Paramètres → Code d'accès — **seul le SuperAdmin** peut effectuer cette modification.

- **Ajouter un administrateur** : Faire via le formulaire d'ajout d'admin en fournissant le code d'autorisation requis ; le SuperAdmin peut bloquer ou modifier des admins.
